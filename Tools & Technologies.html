1 PyTorch

PyTorch includes two data-handling routines. They are torch.utils.data.DataLoader and torch.utils.data.Torch.utils.data.Torch.utils.data.Torch.utils.data.Tor Dataset. Dataset converts the variable to a tensor, and DataLoader wraps an iterable around it. PyTorch additionally includes libraries for a variety of applications. TorchText, TorchVision, and TorchAudio are the three. Each of them has its own set of data. This course will make use of a TorchVision dataset. There are two variables in every TorchVision: transform and target transform. Transform effects samples, whereas target transform modifies labels.

2 Numpy

NumPy is a Python library that can conduct a broad number of different operations on arrays. It adds complicated data structures for fast arrays and matrix calculation, as well as a vast library of large mathematical functions that work on these arrays and matrices, to Python.

3 Flask API

Flask was used to create the web application because Flask is a small and lightweight Python web structure that offers useful features and tools that make it easier to construct Python-based online applications
